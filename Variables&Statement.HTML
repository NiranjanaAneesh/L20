Addition

Addition is combining two or more numbers together to get a new total. The symbol used in addition problems looks like a cross, +, and is called a plus sign. The total for addition is called the sum. Other common words used in addition problems are add, plus, combine, together, total, in all, and increase.

JavaScript Statements

A JavaScript statement is a piece of code that tells a computer or browser to perform an action. JavaScript statements can be simple or complex, and can include values, operators, expressions, keywords, and comments. Here are some examples of JavaScript statements: Assignment statements: Assign values to variables that have already been declared. For example, myName = "Jane". Declaration statements: Create variables. For example, let bestColor. Expression statements: Use almost any expression as a statement. For example, console.log("Hello"). Conditional statements: Control program flow. For example, if. New statements: Create new objects. For example, let myInstance = new MyObject(params). JavaScript statements are executed in the order they are written in a program. Semicolons (;) are used to separate statements, but they are not required at the end of each statement.
